Pumili ng clients para magpadala ng reminders

Awtomatikong nagpapadala ng SMS araw-araw batay sa mga rules na ito. Hindi na kailangan pang manual na i-click — basta naka-deploy na ang app.

Day-before reminder
Nagpapadala ng SMS sa lahat ng clients na may appointment bukas sa 10:00 AM. Hindi ire-resend kung "REMINDER SENT" ay Done na sa Monday board.
Filter: Appointment status = For confirmation OR Confirmed · Cron: 0 2 * * * (UTC = 10AM PH)
Day-of reminder
Nagpapadala ng SMS sa mga Confirmed clients sa araw mismo ng kanilang appointment, 8:00 AM.
Filter: Appointment status = Confirmed · Cron: 0 0 * * * (UTC = 8AM PH)

📌 Para gumana ang auto rules — i-deploy sa Vercel

Sundin ang mga steps sa README.md na kasama sa files na ito. Libre ang Vercel at automatic na mag-ri-run ang SMS araw-araw.

1
I-download ang project files at i-upload sa GitHub (libre ang account)
2
Mag-sign up sa vercel.com gamit ang GitHub account mo
3
I-connect ang GitHub repo sa Vercel — i-set ang MONDAY_TOKEN at SEMAPHORE_KEY sa environment variables
4
I-deploy! Automatic na mag-se-send ng SMS araw-araw ng 10AM.
